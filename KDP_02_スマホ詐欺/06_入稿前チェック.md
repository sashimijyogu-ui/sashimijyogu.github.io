## チェック項目

### EPUB
- [ ] 表紙画像が内包されているか（Calibreで開いて1ページ目に表紙が出る）
- [ ] 目次（nav）が正しく生成されているか（序章〜終章＋奥付が並ぶ）
- [ ] 文字化けがないか（日本語が□や?にならない＝UTF-8で保存）

### 表紙
- [ ] 1600×2560px以上か
- [ ] JPEG形式・RGBカラーか（CMYKは不可）
- [ ] サムネイルで「スマホ」「詐欺」「7つの設定」が読めるか

### メタデータ

- [ ] 成人向けフラグ：正しく設定されているか（→ 本書は「いいえ」）
- [ ] 価格：99〜20,000円の範囲か（→ ¥500 ✅）
- [ ] 著作権：「自分が著作権を保有」を選択

### 最終プレビュー
- [ ] Calibre等でEPUBを実際にプレビューした

## 問題があった場合の修正方法（早見表）

| チェックでNGだったら | 対処 |
|---|---|
| 表紙が内包されていない | `--epub-cover-image=cover.jpg` を付けて再変換。パスを確認 |
| 目次が出ない | `--toc --toc-depth=1` を付ける。章頭が `#`（H1）か確認 |
| 文字化け | 原稿をUTF-8で保存し直して再変換 |
| 表紙が小さい/CMYK | Canvaで「JPG・1600×2560px・RGB」で書き出し直す |
| タイトルが36字超 | サブタイトルを短縮（例：「7つの設定」を残し説明的部分を削る） |
| 説明文が長い | HTMLタグも文字数に含むため、段落を1つ削る |
