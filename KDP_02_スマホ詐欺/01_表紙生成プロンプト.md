# 表紙生成プロンプト

- **タイトル**：スマホで詐欺に遭わない本
- **サブタイトル**：シニアが今すぐやるべき7つの設定
- **ジャンル・雰囲気**：実用書／防犯ガイド風。クリーンで信頼感があり、シニアにも見やすい「安心感のある」トーン。大きな文字・高コントラスト・優しい色味。

## Canva制作手順

### 1. テンプレートを開く
1. Canvaにログイン →「デザインを作成」をクリック
2. 「カスタムサイズ」を選び、**幅 1600 px × 高さ 2560 px** を入力して作成
   - （検索で「Kindle 表紙」テンプレートを選んでもOK。比率1.6:1）
3. 上で生成した背景画像を「アップロード」して全面に配置

### 2. 推奨フォント・配色
- **タイトル**：太めのゴシック体（`Noto Sans JP Bold` / `源ノ角ゴシック Heavy` / `M PLUS 1p Bold`）
- **配色**：
  - 背景：ライトブルー〜ホワイト（#EAF4FB → #FFFFFF）
  - メイン文字：濃紺（#1B3A5B）＝信頼感
  - アクセント（「7つの設定」など）：オレンジ（#F5A623）or 緑（#2E9E5B）
- シニア向けなので、**文字は大きく・コントラスト強め**が鉄則

### 3. タイトル文字のレイアウト案
```
┌────────────────────┐
│                              │
│   スマホで              │ ← 上段：大きく濃紺
│   詐欺に遭わない本   │ ← メインタイトル（最大級）
│                              │
│   〔スマホ＋盾のイラスト〕 │ ← 中央
│                              │
│  シニアが今すぐやるべき  │ ← サブタイトル（やや小さく）
│   ７つの設定            │ ← 「7」を大きく色付きで強調
│                              │
│              村上 健一  │ ← 下段右：著者名
└────────────────────┘
```
- 「**7**つの設定」の数字「7」だけオレンジで大きくして、ひと目で内容が伝わるように
- タイトルは2〜3行に折り返し、1行あたりの文字を大きく保つ

## 表紙チェックリスト

- [ ] 1600×2560px（縦長・比率1.6:1）
- [ ] JPEG形式、RGBカラー
- [ ] タイトル・サブタイトル・著者名が入っている
- [ ] サムネイル（小さい表示）でも「スマホ」「詐欺」「7つの設定」が読める
- [ ] 文字と背景のコントラストが十分（シニアが見やすい）
- [ ] 怖すぎず、安心感のあるトーンになっている
- [ ] （成人向けではないため露骨表現の懸念なし）
